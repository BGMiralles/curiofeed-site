curiofeed

An automated content pipeline operated by a single individual.

curiofeed selects topics from public YouTube trend data, writes and narrates short educational videos about science, history and curiosities, and uploads them to its operator's own YouTube channel, The Quiet Unknown.

Not a public service. curiofeed is not available to anyone else. It runs locally on its operator's machine, authenticates as that operator, and uploads only to that operator's own channel. There are no user accounts and no sign-up: the application has no users other than the person who runs it.

AI-generated content

All visuals and narration are AI-generated. Every upload is declared as synthetic media through the YouTube API (status.containsSyntheticMedia = true), is uploaded with private visibility, and is reviewed by a human before any public release.

Use of YouTube API Services

curiofeed uses YouTube API Services. It reads public trend signals through the YouTube Data API to choose topics, and uploads videos to its operator's own channel using a single OAuth scope, https://www.googleapis.com/auth/youtube.upload. It does not read, modify or delete data belonging to any other account or channel, and it never resells or redistributes YouTube API data.

Its use of YouTube API Services is subject to the YouTube Terms of Service, and information handled through Google APIs is governed by the Google Privacy Policy. Access can be revoked at any time at https://myaccount.google.com/connections.
